## Функционал

Проект представляет собой систему отслеживания зоны интереса (столик) на
предмет активности. На видео вручную выбирается зона интереса, затем, в
реальном времени отслеживается нахождение людей в кадре, в случае если
человек находится в выбранной зоне интереса программа детектирует событие
и записывает время события.

## Структура проекта

- main.py - исходный код программы

## Запуск

Для запуска программы требуется установить зависимости python
и запустить скрипт main.py, передав ему в качестве параметра
входной видеофайл

1. установка зависимостей: ```pip install -r requirements.txt ```
2. запуск программы: ```python main.py --video input.mp4```
   (убедитесь что в имени файла отсутствуют пробелы, или передавайте параметр в кавычках "input 1.mp4")

так же возможно передать опциональный параметр headless (по-умолчанию true)
которые принимает булевы значения (true/false)и отвечает за то, нужно ли
выводить окно с процессом обработки видеофайла (false - выводить, true - не
выводить)

например: ```python main.py --video input.mp4 --headless false```
